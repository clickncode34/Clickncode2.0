# 🎨 PROMPT TEMPLATE - PLATEFORME E-COMMERCE LUXURY

## 📋 CONTEXTE & OBJECTIF

Tu es un expert en développement web e-commerce. Je veux créer un site internet professionnel et luxe pour vendre [TYPE_PRODUIT] en ligne.

Le site doit être:
- ✅ Moderne et élégant (design premium)
- ✅ Performant (60fps, sans freeze)
- ✅ Sécurisé (paiements + données)
- ✅ Responsive (mobile/tablette/desktop)
- ✅ Sans backend lourd (GitHub Pages friendly)
- ✅ Paiements sécurisés via Stripe

## 🛠️ STACK TECHNOLOGIQUE REQUIS

### Frontend
- **HTML5** avec structure sémantique
- **CSS3** avec:
  - Grid & Flexbox pour layouts
  - CSS Variables pour theming
  - Animations cubic-bezier pour fluidité
  - Media queries responsives
- **Vanilla JavaScript ES6** (pas de framework lourd)
  - Pas de jQuery, pas de React
  - Performances natives optimales

### Hébergement & Paiements
- **GitHub Pages** (HTTPS automatique, CDN global)
- **Stripe** pour paiements (buy.stripe.com hosted forms)
- **Service Worker** pour PWA (offline capability)
- **No Backend** (site 100% statique)

### Ressources Externes
- **Google Fonts** (Montserrat + 1 serif pour headings)
- **Font Awesome** ou SVG icons (léger)
- **Images optimisées** (WebP si possible, JPEG fallback)

## 🎯 ARCHITECTURE GÉNÉRALE

### Structure de Fichiers
```
/
├── index.html          # Page principale (tout en une)
├── style.css           # Styles master (~2000-2500 lignes)
├── script.js           # Interactivité (~500-800 lignes)
├── service-worker.js   # PWA offline support
├── manifest.json       # PWA metadata
├── robots.txt          # SEO bot management
├── sitemap.xml         # URL listing for search engines
├── SECURITY.md         # Documentation sécurité
└── images/             # Assets optimisées
```

### Organisation HTML - Sections
1. **Header/Navigation** - Logo + menu sticky
2. **Hero Section** - Banneau attrayant avec CTA
3. **Produits/Catalog** - Grid responsive
4. **À Propos/Brand Story** - Positioning premium
5. **Galerie/Portfolio** - Images de qualité
6. **Testimonials/Reviews** - Preuve sociale
7. **Newsletter** - Email signup
8. **Contact Section** - 3 boutons sociaux + formulaire
9. **Footer Premium** - 4 colonnes (Brand, Links, Info, Contact)

## 🎨 DESIGN & STYLING

### Palette Couleurs (Luxury Theme)
```css
--color-primary: #D4AF37;      /* Gold */
--color-dark: #0F0F0F;         /* Deep black */
--color-accent: #2A2A2A;       /* Dark gray */
--color-light: #F5F5F5;        /* Off-white */
--color-white: #FFFFFF;
```

### Typographie
- **Body**: Montserrat (300-400 weight) 16px
- **Headings**: Serif font (700-900 weight) ou Montserrat bold
- **Accents**: 12-14px, letterspacing augmenté

### Animations & Transitions
- ✅ Hover effects fluides (0.2-0.3s)
- ✅ Page load animations (fade-in, slide-up)
- ✅ Scroll animations (parallax, fade on scroll)
- ✅ Particles/Stars (12 max, GPU accelerated)
- ✅ Cubic-bezier curves pour élégance: `cubic-bezier(0.25, 0.46, 0.45, 0.94)`

### Responsiveness Breakpoints
```css
Mobile: < 768px
Tablet: 768px - 1024px
Desktop: > 1024px
Ultra-wide: > 1440px
```

## ⚡ PERFORMANCE REQUIREMENTS

### Optimisations Obligatoires
1. **Réduire particles** - Max 12 particles actifs
2. **Throttle scroll events** - 16ms minimum (60fps)
3. **requestAnimationFrame** - Pour animations fluides
4. **GPU acceleration** - will-change, transform, backface-visibility
5. **Lazy loading** - Images load au scroll
6. **Minify CSS/JS** - Production ready
7. **Image compression** - Moins de 200KB total images
8. **Pas de loops infinies** - Vérifier console for warnings

### Cibles Performance
- ✅ First Contentful Paint < 2s
- ✅ Time to Interactive < 3s
- ✅ Lighthouse Score > 85
- ✅ 60fps constant scrolling
- ✅ Pas de jank/freeze

## 📊 EXEMPLE: CONVERSION DU TEMPLATE

Pour adapter ce template à un nouveau produit:

1. **Remplacer le nom**: "Bijoux2luxe" → "[NOUVEAU_SHOP]"
2. **Remplacer la couleur**: "#D4AF37" → "[COULEUR_MARQUE]"
3. **Remplacer le produit**: "Colliers/Gourmettes" → "[VOTRE_PRODUIT]"
4. **Remplacer les images**: Galerie bijoux → Galerie votre produit
5. **Remplacer les liens Stripe**: buy.stripe.com/old → buy.stripe.com/new
6. **Remplacer les textes**: Descriptions bijoux → Descriptions votre produit
7. **Remplacer les couleurs sociales**: (Bleu/Jaune/Cyan) → (Vos couleurs)
8. **Remplacer le footer**: Infos bijoux → Infos entreprise
9. **Tester** sur mobile/desktop/paiement
